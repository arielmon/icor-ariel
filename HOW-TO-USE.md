# ICOR Life OS — How to Use

## Quick Reference

| I want to... | Do this |
|---|---|
| Capture something quickly | Send Claude: "add to inbox: [thing]" |
| Start my day | Run `/today` or ask: "what's on my plate today?" |
| Process my inbox | Run `/control` or ask: "let's process my inbox" |
| Write a brain dump | Run `/brain-dump` or ask: "brain dump: [thoughts]" |
| Check my goals | Run `/okrs` or ask: "how are my OKRs going?" |
| Do my weekly review | Run `/review` on Sundays |
| Save knowledge | Ask: "save this to my knowledge base: [topic]" |

## The ICOR Cycle

Your system runs on a daily and weekly rhythm:

### Daily (15 min total)
**Morning (5 min):**
- Open Claude (terminal, Cowork, or phone)
- Run `/today` — Claude shows your top tasks, calendar, and any overdue items
- Pick your Top 3 "Must" tasks for the day

**During the day:**
- Capture anything that comes up: "add to inbox: [thing]"
- Work on tasks based on your energy level, not a rigid schedule
- Brain dump anytime: "brain dump: [thoughts]"

**Evening (5 min):**
- Tell Claude what you completed: "done: [task]" or "mark [task] as done"
- Capture anything for tomorrow
- Optional: quick journal entry

### Weekly (45 min — pick a consistent day, e.g. Sunday)
- Run `/review`
- Claude walks you through:
  1. Processing inbox to zero
  2. Reviewing all active tasks and projects
  3. Checking calendar for next 2 weeks
  4. Scoring your OKRs (Green/Yellow/Red)
  5. Identifying neglected life areas
  6. Setting Top 3 for next week

### Monthly (1 hour — last day of month)
- Review OKR progress in detail
- Clean up knowledge base (archive what's stale)
- Adjust priorities for next month

### Quarterly (2 hours — end of quarter)
- Set new OKRs (3-5 max)
- Life area assessment across all areas
- Major system adjustments if needed

## How to Capture from Mobile

**Option A: Claude App**
1. Open Claude on your phone
2. Make sure this project/workspace is selected
3. Type naturally: "add to inbox: schedule dentist appointment"
4. Claude confirms with a one-liner

**Option B: Remote Control (Claude Code)**
1. On your computer, run: `claude remote-control`
2. Scan the QR code with your phone
3. You now have full access to all slash commands from mobile

## Task Priority System

| Priority | Meaning | When to use |
|---|---|---|
| **Must** | Non-negotiable | Tied to an OKR, has a hard deadline, or consequences if missed |
| **Should** | Important, flexible | Moves your goals forward but timing is flexible |
| **Could** | Nice to have | Do when you have extra energy/time |

## Life Area Tags

Use these to categorize everything:
- `#work` — Career, client tasks, professional development
- `#health` — Exercise, nutrition, medical, mental health
- `#finance` — Budget, bills, investments, savings
- `#learn` — Courses, books, skills, tutorials
- `#social` — Relationships, events, family, friends
- `#home` — Household, errands, maintenance
- `#projects` — Side projects, hobbies, creative work

## Knowledge Base (Second Brain)

When you find something worth keeping that isn't a task:
- "Save to knowledge base: [topic] — [content/link]"
- Claude files it in the right PARA category:
  - **Projects/** — Active efforts with a deadline
  - **Areas/** — Ongoing responsibilities
  - **Resources/** — Future reference by topic
  - **Archives/** — Done or inactive

## OKR Basics

- Set 3-5 objectives per quarter (not per area)
- Each objective has 2-3 measurable key results
- Score weekly: Green (on track), Yellow (at risk), Red (off track)
- 70% completion = success — these are stretch goals
- Review during weekly `/review`

## Tips

1. **Capture everything, decide later** — don't filter at capture time
2. **Energy matching > scheduling** — do hard work when focused, easy work when tired
3. **Weekly review is sacred** — skip it and the system decays
4. **Start with just inbox + tasks** — add OKRs and journal after week 2
5. **Talk to Claude naturally** — you don't need exact commands
