Weekly review — walk me through the full ICOR cycle.

Guide the user through a structured weekly review (~45 min). Read `Reviews/weekly/template.md` for the template structure, then walk through each section:

**1. INPUT — Process Inbox**
- Run the inbox processing flow (same as /control)
- Goal: inbox to zero

**2. CONTROL — Review Active Work**
- Read `Tasks/tasks.md` — review each task: still relevant? done? stuck?
- Check Google Calendar for next 2 weeks using `mcp__claude_ai_Google_Calendar__gcal_list_events`
- Identify any neglected life areas (#work #health #finance #learn #social #home #projects)

**3. OUTPUT — Celebrate Wins**
- Ask: "What did you accomplish this week?"
- Log wins

**4. REFINE — Score & Adjust**
- Read `OKRs/current-quarter.md` and score each key result: Green/Yellow/Red
- Ask: "What worked well? What caused friction?"
- Identify system adjustments

**5. Plan Next Week**
- Ask: "What are your Top 3 priorities for next week?"
- Update tasks accordingly

Save the review to `Reviews/weekly/` with today's date as filename.
